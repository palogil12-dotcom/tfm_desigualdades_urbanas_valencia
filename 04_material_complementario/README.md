# Material complementario

Esta carpeta esta reservada para el material que no conviene mezclar con el anexo principal del TFM.

## Estructura

- `datos_brutos/`: fuentes originales o descargas pesadas.
- `resultados_intermedios/`: tablas de trabajo y versiones previas.
- `cartografia/`: geojson, shapefiles exportados o mapas auxiliares.
- `figuras/`: graficos secundarios o imagenes no esenciales.
